# 🚀 UserSphere - Sistema de Gestão de Projetos e Tarefas  

## 📖 Sobre o Projeto  
O **UserSphere** é um sistema completo para **gestão de projetos e tarefas**, desenvolvido para **organizar, acompanhar e otimizar a produtividade** de indivíduos e equipes. Com uma interface intuitiva, permite criar projetos, adicionar tarefas e acompanhar o progresso de cada atividade em tempo real.  

## 🛠 Tecnologias Utilizadas  
- **Frontend:** Vue.js 3 + Vue Router + Bootstrap 5  
- **Backend:** Laravel + MySQL  
- **API:** Axios para comunicação entre frontend e backend  
- **Autenticação:** JWT para segurança dos usuários  

## ✨ Funcionalidades  
✅ **Cadastro e Login de Usuários** com autenticação JWT  
✅ **Gerenciamento de Projetos** (CRUD completo)  
✅ **Gerenciamento de Tarefas** dentro de cada projeto  
✅ **Edição e Atualização** de Tarefas diretamente pelo sistema  
✅ **Alteração de Status** das tarefas (**Pendente, Em andamento, Concluído**)  
✅ **Dashboard** para melhor visualização dos projetos e tarefas
